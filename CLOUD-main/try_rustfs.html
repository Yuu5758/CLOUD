RustFS Documentation

Complete Quick Start Guide and SDK integrations to integrate your application with RustFS.

🚀 Quick Start with AWS CLI

RustFS is 100% compatible with the AWS S3 REST protocol. You can use standard client SDKs or the AWS CLI. Simply configure your AWS profile or supply custom endpoint credentials.

Terminal / Bash 
# Configure AWS Credentials (only needed once)
aws configure set aws_access_key_id rustfsadmin
aws configure set aws_secret_access_key rustfsadmin

# List buckets
aws s3 ls --endpoint-url http://127.0.0.1:9000

# Create a new bucket
aws s3 mb s3://my-bucket --endpoint-url http://127.0.0.1:9000

# Upload a file
aws s3 cp document.pdf s3://my-bucket/documents/ --endpoint-url http://127.0.0.1:9000

🐍 Python Integration (Boto3)

Connecting your Python backend is extremely straightforward. Make sure to specify the custom local endpoint URL.

Python (boto3) 
import boto3

# Initialize the S3 client for local RustFS
s3_client = boto3.client(
    's3',
    endpoint_url='http://127.0.0.1:9000',
    aws_access_key_id='rustfsadmin',
    aws_secret_access_key='rustfsadmin',
    region_name='us-east-1'
)

# List all buckets
response = s3_client.list_buckets()
for bucket in response['Buckets']:
    print(f"Bucket: {bucket['Name']}")

🟢 Node.js / JavaScript SDK

For JavaScript web applications or Node.js backends, load the official AWS SDK v3.

Node.js (AWS SDK v3) 
const { S3Client, ListBucketsCommand } = require("@aws-sdk/client-s3");

const client = new S3Client({
  endpoint: "http://127.0.0.1:9000",
  credentials: {
    accessKeyId: "rustfsadmin",
    secretAccessKey: "rustfsadmin"
  },
  forcePathStyle: true, // Crucial for standalone S3 systems
  region: "us-east-1"
});

async function run() {
  const response = await client.send(new ListBucketsCommand({}));
  console.log(response.Buckets);
}

🦀 Rust Integration (aws-sdk-s3)

Given that RustFS is written in Rust, leveraging the official AWS SDK for Rust yields peak performance.

Rust (Cargo.toml + main.rs) 
// Setup connection client
let config = aws_config::from_env()
    .endpoint_url("http://127.0.0.1:9000")
    .load().await;

let s3_client = aws_sdk_s3::Client::new(&config);

// Retrieve buckets list
let resp = s3_client.list_buckets().send().await?;
for bucket in resp.buckets() {
    println!("Bucket: {:?}", bucket.name());
}
